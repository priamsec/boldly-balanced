📖 Boldly Balance API

Public API for accessing Boldly Balance blog content.

Base URL

https://blog.boldlybalance.life/api

Endpoints

GET /posts

Returns all blog posts sorted by date (newest first).

Response

{"posts": [{ "slug": "...", "title": "...", "category": "...", ... }]}
GET /posts/{slug}

Returns a single post by its slug.

GET /info

Returns API information.

Example Usage

fetch('https://blog.boldlybalance.life/api/posts.json')
  .then(res => res.json())
  .then(data => console.log(data.posts));
