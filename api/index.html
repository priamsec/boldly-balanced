🔌 Boldly Balance API

A simple, read-only JSON API for accessing Boldly Balance blog content.

Base URL

https://boldlybalance.life/api

Endpoints

GET /info.json

API information and metadata.

GET /posts.json

List all posts with summaries (112 posts).

GET /categories.json

List all categories with post counts.

GET /posts/{slug}.json

Get a specific post by slug (e.g., /burnout-recovery-guide.json).

Example Usage

# Get all posts
curl https://boldlybalance.life/api/posts.json

# Get categories
curl https://boldlybalance.life/api/categories.json

# Get specific post
curl https://boldlybalance.life/api/posts/burnout-recovery-guide.json
